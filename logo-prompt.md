# Examine CBT Platform - Logo Design Prompt

## Design Brief

Create a geometric, minimal logo for "Examine" - an AI-powered CBT exam preparation platform for JAMB, WAEC, and NECO students in Nigeria.

## Style Reference

- Similar to Warp terminal's logo: clean geometric shapes, modern, tech-forward
- Minimal and sophisticated
- Works well at small sizes (favicon, navigation)
- Scalable vector format (SVG)

## Design System Colors

**Primary Gradient:**

- Start: #0ea5e9 (Cyan/Blue)
- End: #0284c7 (Deeper Blue)

**Secondary Accent:**

- #d946ef (Purple/Magenta)

**Alternative Gradients:**

- Primary to Secondary: #0ea5e9 → #d946ef
- Full spectrum: from-primary-500 to-secondary-500

## Technical Requirements

- **Format:** PNG, SVG (inline, clean code)
- **Size:** Optimized for 32x32px to 200x200px
- **Colors:** Use CSS variables or direct hex codes from palette
- **Shapes:** Prefer `<path>`, `<polygon>`, `<circle>` elements
- **Gradient:** Include `<linearGradient>` or `<radialGradient>` definitions
- **No text:** Logo mark only (wordmark separate)

## Design Principles

1. **Geometric Precision:** All shapes should be mathematically precise
2. **Minimal Complexity:** Maximum 3-5 main shapes
3. **Recognizable:** Identifiable even at 16x16px
4. **Modern:** Tech-forward, not traditional education imagery
5. **Gradient Flow:** Smooth transition from blue to purple
6. **Negative Space:** Use cleverly (like FedEx arrow)

## Avoid

- ❌ Mortarboard/graduation caps (too cliché)
- ❌ Traditional books (too literal)
- ❌ Pencils/pens (overused in education)
- ❌ Light bulbs (unless abstract/geometric)
- ❌ Overly complex illustrations
- ❌ Realistic/photographic styles

## Brand Personality

- **Smart:** AI-powered, intelligent platform
- **Modern:** Tech-forward, not traditional classroom
- **Trustworthy:** Reliable exam preparation
- **Aspirational:** Success, achievement, excellence
- **Youthful:** Appeals to students (16-25 age range)

## Logo Variations Needed

1. **Full Color Gradient** - Primary usage
2. **Single Color (Blue)** - For limited color contexts
3. **White** - For dark backgrounds
4. **Black** - For print/monochrome
